# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its code of conduct. The full text is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — submitting issues, pull requests, comments, or any other contribution — you agree to uphold these standards.

## Reporting

Concerns about behaviour in this community can be reported privately to **conduct@klaro.so**. Reports are reviewed by the Klaro maintainers and treated confidentially. The maintainers may consult external mediators where appropriate.

Reports include, at minimum:

- A description of what happened
- Where and when it happened
- Whether the incident is ongoing
- Any supporting links, screenshots, or context

The maintainers will respond within five business days with the next steps.

## Enforcement

The maintainers may take any action they deem appropriate — including warning the contributor, requiring a public or private apology, temporary or permanent ban from project spaces — in response to behaviour that violates this code.

Decisions are documented internally. Sanctioned contributors may appeal in writing to **conduct@klaro.so** within thirty days.

## Scope

This code applies to all project spaces — this repository, the issue tracker, pull requests, discussion threads, official chat channels, in-person and virtual events organised by the project, and any other forum where someone is representing Klaro.
