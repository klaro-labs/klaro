2 · The buyer's pay flow — apps/web/components/klaro/PayWithUSDC.tsx:157–192 ↗

// Live path — full EIP-712 sign → approve → acceptAndPay.
const sig = await signTypedDataAsync({            // EIP-712 acceptance
  domain: { name: "Klaro Invoice", version: "1",
            chainId: arcTestnet.id, verifyingContract: INVOICE_ESCROW_ADDRESS },
  types: ACCEPTANCE_EIP712_TYPES, primaryType: "InvoiceAcceptance",
  message: { invoiceId, vendor, token, amount, dueAt, metadataHash, splitsHash },
});
await writeContractAsync({ address: token, abi: ERC20_ABI,
  functionName: "approve", args: [INVOICE_ESCROW_ADDRESS, amount] });
const txHash = await writeContractAsync({ address: INVOICE_ESCROW_ADDRESS,
  abi: INVOICE_ESCROW_ABI, functionName: "acceptAndPay", args: [invoiceId, sig, address] });

The buyer signs an EIP-712 acceptance, approves the escrow, and pays in one click — the payment is a cryptographic acceptance bound to the exact invoice, not a raw transfer.

Integration demonstration — Circle infrastructure powering the product, live

The exact user flow the video demos, narrated step by step — every "→ tx" opens the real transaction on the explorer so you can verify it ran, on-chain, with status = success.

Vendor creates + publishes the invoice on-chain
A hosted page at /i/<id> is generated and InvoiceEscrow.createInvoice writes it to Arc (CREATED). → publish tx
Buyer pays in USDC on Arc
The buyer connects a wallet, signs the EIP-712 acceptance, approves, and pays — acceptAndPay moves USDC into escrow (PAID). → pay tx
Operator daemon screens + settles
OFAC sanctions + KYB (GREEN) pass → the operator calls settle, releasing USDC to the vendor (SETTLED). Compliance-cleared and work-verified. → settle tx
A receipt is anchored — publicly verifiable
An audit-grade receipt is written to the AuditReceipt contract. verify returns true; anyone can check it on-chain or via the public API — no personal data, only hashes + addresses + status. → receipt (verify=true) · → public API
Or the buyer pays from another chain (Circle CCTP V2)
USDC is burned on Base Sepolia, attested by Circle Iris, and the operator mints native USDC to the vendor on Arc — no bridge, no wrapped token. → burn tx (Base) · → mint tx (Arc)
