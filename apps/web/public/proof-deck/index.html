Codebase walkthrough — where Circle technologies are implemented

  • USDC settlement:packages/contracts/src/InvoiceEscrow.sol — ERC-20 (USDC, 6-decimal Arc interface 0x3600…0000) in escrow; acceptAndPay / settle state machine (PAID → SETTLED).
  • Wallet pay flow:apps/web/components/klaro/PayWithUSDC.tsx:97–192 — EIP-712 acceptance sign (signTypedDataAsync) → approve → acceptAndPay via wagmi.
  • CCTP V2 pay-in:apps/web/scripts/cctp-payin-proof.mjs:29–37 — depositForBurn on Base (TokenMessenger 0x8FE6…2DAA) → Circle Iris attestation → operator receiveMessage mint on Arc (MessageTransmitter 0xE737…E275).
  • Circle Modular Wallets:apps/web/lib/circleModularWallet.ts + components/klaro/CircleWalletButton.tsx — passkey-backed smart-account onboarding (no seed phrase).
  • Compliance screening:apps/daemon/src/ofac.ts — OFAC SDN sanctions gate before funds release; apps/daemon/src/sumsub.ts — KYB (GREEN) gates settle. Compliance-cleared and work-verified.
